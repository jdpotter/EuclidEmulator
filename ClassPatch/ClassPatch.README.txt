Install the patched version of CLASS that is required for EuclidEmulator from 

https://github.com/miknab/ClassPatch/archive/master.tar.gz

On Linux, this is as easy as executing

wget https://github.com/miknab/ClassPatch/archive/master.tar.gz

Extract the tar ball 

tar -xvg master.tar.gz

cd into the newly generated directory and in there type "make". NOTICE: This will not
just build but also install the classee.so to your python's site-package directory.
